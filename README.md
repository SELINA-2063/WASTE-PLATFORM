# Waste to Resource Exchange Platform (W2REP)

## 🌿 Project Overview
A web-based platform where sellers can post recyclable waste materials and buyers can send requests to purchase them.

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL

## ⚙️ Features
- User Registration & Login (Buyer, Seller, Admin)
- Waste Listing & Search
- Buyer Request System
- Seller Dashboard
- Buyer Dashboard
- Admin Dashboard

## 🚀 How to Run
1. Clone the repository
2. Run `npm install`
3. Start MySQL and create database `w2rep_db`
4. Run `node server.js`
5. Open `index.html` in browser
